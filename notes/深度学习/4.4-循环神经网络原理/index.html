4.4 循环神经网络原理

循环神经网络，Recurrent Neural Network（RNN），提出时间很早，上个世纪90年代就有了，其变体在14年左右才出现。在RNN出现之前，在处理语言时遇到了瓶颈：

  • NLP领域的主流模型（如N-grams、词袋模型Bag-of-Words）：统计模型计算量非常大非常复杂；词袋等词嵌入模型没有考虑token的语序关系，比如“人咬狗”和“狗咬人”的语义完全不同。
  • 传统的神经网络（如前馈神经网络FNN，或者叫多层感知机MLP）：固定长度的输入输出，而且没有记忆。前馈神经网络认为特征（token）之间是独立同分布的，对具有时序属性的数据没法很好地处理。

当然，RNN也有不同的结构，不一定非要是第1节中所讲的那样。当前时间的输入可以是上一个时间点的输出，即Xt=yt−1X_t=y_{t-1}Xt​=yt−1​。这样的结构可以用于文本生成、时间序列的预测等任务。接下来，使用RNN来实现对f(x)=sin⁡xf(x)=\sin xf(x)=sinx函数变化趋势的预测。首先在0~100内等分得到2000个点，前1000个点作为训练集，用来预测后面1000个点，采用自回归（auto-regression）的方式预测

[!NOTE]

自回归预测：假设有如下周期的数据，令seq_len=5（滑动窗口大小）
① 1 2 3 2 1 0 -1 -2 -3 -2 -1 0 1 2 3，通过前面滑动窗口中的5个数字预测下一个数字为2.2，将2.2加入到列表尾部，滑动窗口向后移动一位（这里用下划线代替）；
② 1 2 3 2 1 0 -1 -2 -3 -2 -1 0 1 2 3 2.2，同①一样的操作，预测下一个数字为1，将预测值添加到尾部，然后滑动窗口向后滑动一位；
③ 1 2 3 2 1 0 -1 -2 -3 -2 -1 0 1 2 3 2.2 1
文本生成就是通过这样的方式实现的。

由图可知，RNN学到了正弦函数的周期性。与此同时，这里做了一组对比试验，在不同的seq_len的情况下，模型的预测结果，可以发现在seq_len较大的情况下，在时间后期出现了长期预测的漂移（Drift）：随着预测时间变长，红色的虚线向左或者右偏移。这是因为在生成式预测中，长时依赖导致每一步的微小误差都会被累积，并作为下一步的输入，导致误差被放大。

可以发现多层感知机对于单个特征的输入完全不能提取出特征。如果将输入改为和RNN一样的，也就是将前seq_len个函数值作为输入特征，又会得到怎样的结果呢？令上述MLP的input_size=seq_len，可以得到比RNN还要完美的预测图，这里就不将结果展示了（红色虚线和蓝色真实的线几乎完美重合）。这是否说明RNN在预测这一块的能力很弱？

实际上，RNN本来就是为处理自然语言而诞生的，在本例中，sin⁡x\sin xsinx具有明显的周期性，只要输入的特征足够多，MLP也能通过固定长的序列作为输入特征，更容易找到函数的周期性。在一般任务中，比如股票、天气或者轨迹预测，实际上RNN也不一定能够表现得非常好。比如天气，影响的因素有温度、湿度、降水量等等，这些特征作为输入向量XtX_tXt​，在NLP中对应的就是词嵌入向量，那么是否能够像生成文本那样预测出未来几天或者十几天的天气呢？显然非常困难，因为并不能明确t+n (n=1,2,⋯ )t+n\ (n=1,2,\cdots)t+n (n=1,2,⋯)时刻输入的向量，这些值需要当天测量。之所以文本生成能够生成一长串的语句，“预测”到后面几十上百个时间点的token，是因为所有的词嵌入向量已经存储在词表中了，也就是说知道了yt+1y_{t+1}yt+1​，那么Xt+2X_{t+2}Xt+2​也能够明确。因此对于循环神经网络，我们仅讨论NLP相关的任务。

4.5 RNN变体

1 LSTM

Long Short-Term Memory, 长短时（长短期）记忆网络，由Sepp Hochreiter和Jürgen Schmidhuber在1997年提出，是为了解决传统RNN“健忘”（梯度消失）的问题。下图是博客Understanding LSTM中的经典图片，这里就仿照画了一个（右图）。相比之下，左图是经典RNN结构，LSTM的输入输出维度是类似的。

      

首先规范数学表达，这里的特征用行向量表示，是为了和PyTorch编程保持一致。假设有hhh个隐藏单元，即hidden_size = h；批次大小为nnn，即batch_size = n；输入维度为ddd，即feature_size = d。同RNN一样，LSTM只是替换了细胞单元的内部结构，输入输出的大小还是一样的：Xt∈Rn×d\boldsymbol{X}_t\in \mathbb{R}^{n\times d}Xt​∈Rn×d，Ht−1∈Rn×h\boldsymbol{H}_{t-1}\in\mathbb{R}^{n\times h}Ht−1​∈Rn×h，时间步ttt的门以及细胞中的单元，公式中的σ\sigmaσ是Sigmoid函数：

  • Input Gate（图中把候选记忆元Candidate memory cell也框进去了）

It=σ(XtWxi+Ht−1Whi+bi), ∈Rn×h \boldsymbol{I}_t=\sigma(\boldsymbol{X}_t\boldsymbol{W}_{xi}+\boldsymbol{H}_{t-1}\boldsymbol{W}_{hi}+\boldsymbol{b}_i),\ \in\mathbb{R}^{n\times h} It​=σ(Xt​Wxi​+Ht−1​Whi​+bi​), ∈Rn×h

  • Forget Gate

Ft=σ(XtWxf+Ht−1Whf+bf), ∈Rn×h \boldsymbol{F}_t=\sigma(\boldsymbol{X}_t\boldsymbol{W}_{xf}+\boldsymbol{H}_{t-1}\boldsymbol{W}_{hf}+\boldsymbol{b}_f),\ \in\mathbb{R}^{n\times h} Ft​=σ(Xt​Wxf​+Ht−1​Whf​+bf​), ∈Rn×h

  • Output Gate

Ot=σ(XtWxo+Ht−1Who+bo), ∈Rn×h \boldsymbol{O}_t=\sigma(\boldsymbol{X}_t\boldsymbol{W}_{xo}+\boldsymbol{H}_{t-1}\boldsymbol{W}_{ho}+\boldsymbol{b}_o),\ \in\mathbb{R}^{n\times h} Ot​=σ(Xt​Wxo​+Ht−1​Who​+bo​), ∈Rn×h

  • 记忆元，Cell State

Ct=Ft⊙Ct−1+It⊙C~t. \boldsymbol{C}_t=\boldsymbol{F}_t \odot \boldsymbol{C}_{t-1} + \boldsymbol{I}_t \odot \tilde{\boldsymbol{C}}_t. Ct​=Ft​⊙Ct−1​+It​⊙C~t​.

  • 候选记忆元，Candidate memory cell。和3个门的区别就是使用了不同的激活函数。

C~t=tanh⁡(XtWxc+Ht−1Whc+bc), ∈Rn×h \tilde{\boldsymbol{C}}_t=\tanh(\boldsymbol{X}_t\boldsymbol{W}_{xc}+\boldsymbol{H}_{t-1}\boldsymbol{W}_{hc}+\boldsymbol{b}_c),\ \in\mathbb{R}^{n\times h} C~t​=tanh(Xt​Wxc​+Ht−1​Whc​+bc​), ∈Rn×h

  • 隐状态

Ht=Ot⊙tanh⁡(Ct). \boldsymbol{H}_t=\boldsymbol{O}_t\odot \tanh(\boldsymbol{C}_t). Ht​=Ot​⊙tanh(Ct​).

2 GRU

GRU（门控循环单元，gated recurrent unit）在2014年由Cho等人提出，是LSTM的一个简化版，将Cell State和隐藏状态合并。

同样地，规范数学表达：X∈Rn×d\boldsymbol{X}\in\mathbb{R}^{n\times d}X∈Rn×d，H∈Rn×h\boldsymbol{H}\in\mathbb{R}^{n\times h}H∈Rn×h

  • 重置门，Reset gate

Rt=σ(XtWxr+Ht−1Whr+br), ∈Rn×h. \boldsymbol{R}_t=\sigma(\boldsymbol{X}_t\boldsymbol{W}_{xr}+\boldsymbol{H}_{t-1}\boldsymbol{W}_{hr}+\boldsymbol{b}_r),\ \in\mathbb{R}^{n\times h}. Rt​=σ(Xt​Wxr​+Ht−1​Whr​+br​), ∈Rn×h.

  • 更新门，Update gate

Zt=σ(XtWxz+Ht−1Whz+bz), ∈Rn×h. \boldsymbol{Z}_t=\sigma(\boldsymbol{X}_t\boldsymbol{W}_{xz}+\boldsymbol{H}_{t-1}\boldsymbol{W}_{hz}+\boldsymbol{b}_z),\ \in\mathbb{R}^{n\times h}. Zt​=σ(Xt​Wxz​+Ht−1​Whz​+bz​), ∈Rn×h.

  • 候选隐状态

H~t=tanh⁡(XtWxh+(Rt⊙Ht−1)Whh+bh), ∈Rn×h. \tilde{\boldsymbol{H}}_t=\tanh(\boldsymbol{X}_t\boldsymbol{W}_{xh}+(\boldsymbol{R}_t\odot \boldsymbol{H}_{t-1})\boldsymbol{W}_{hh}+\boldsymbol{b}_h),\ \in\mathbb{R}^{n\times h}. H~t​=tanh(Xt​Wxh​+(Rt​⊙Ht−1​)Whh​+bh​), ∈Rn×h.

  • 隐状态

Ht=Zt⊙H~t+(1−Zt)⊙Ht−1, ∈Rn×h. \boldsymbol{H}_t=\boldsymbol{Z}_t\odot \tilde{\boldsymbol{H}}_t +(1-\boldsymbol{Z}_t)\odot \boldsymbol{H}_{t-1},\ \in\mathbb{R}^{n\times h}. Ht​=Zt​⊙H~t​+(1−Zt​)⊙Ht−1​, ∈Rn×h.

3 双向循环神经网络

简单来说，就是将输入的序列正向输入到一个RNN模型中，再将反向输入到另一个RNN模型中，这两个RNN模型相同位置，也就是相同时间步的隐藏层输出连接起来，得到Rn×2h\mathbb{R}^{n\times 2h}Rn×2h维度的向量Ht\boldsymbol H_tHt​，这就是BiRNN的第ttt个时间步的隐藏层输出。

双向循环神经网络必须输入完整的序列进行处理，在NER、摘要、对序分类的任务等方面有优势，同时这也意味着这并不是一个生成模型。
